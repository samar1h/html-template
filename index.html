Getting Started

To use this template for your own project:

  1. Replace icons in the 'icons' folder with your own
  2. Edit the manifest.json file with your app details
  3. Customize the CSS in css/style.css
  4. Update the service worker in sw.js as needed
  5. Modify this HTML structure to fit your needs
